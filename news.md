<h2>
  Release GraphSense 0.4.3
  <small class="text-muted">(2020-05-11)</small>
</h2>

GraphSense 0.4.3 the third minor release of the 0.4 series, has arrived.

This release provides a number of bug fixes, enhancements, and minor features.

#### Dashboard Features / Enhancements

* create and export of TagPacks in various formats
* display transaction details

#### TagPacks

* As a first step into the direction of data sharing and interoperability, GraphSense TagPacks now make use of agreed-upon terms defined as part of the [INTERPOL Darkweb and Cryptocurrencies Taxonomy](https://interpol-innovation-centre.github.io/DW-VA-Taxonomy/).

* GraphSense now provides a dedicated [TagPack Management Tool](https://github.com/graphsense/graphsense-tagpack-tool) for validating and ingesting TagPacks into GraphSense
